## CLI Command Generated

Form submission generates:
```bash
bd create \
  --title "Add rate limiting to authentication endpoints" \
  --type feature \
  --priority p2 \
  --description "## Background\n..." \
  --assignee alice \
  --labels auth,security,backend
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Cmd/Ctrl + Enter` | Submit form |
| `Escape` | Close modal (with unsaved changes warning) |
| `Tab` | Move to next field |
| `Shift + Tab` | Move to previous field |

## Interactions

| Element | Action | Result |
|---------|--------|--------|
| [ร] | Click | Close modal |
| Backdrop | Click | Close modal (with warning if dirty) |
| Type dropdown | Change | Update type, show/hide epic fields |
| [+ Add...] labels | Click | Open label picker |
| Label chip [ร] | Click | Remove label |
| [Cancel] | Click | Close modal |
| [Create Issue] | Click | Validate and submit |
| "Create another" | Check | Keep modal open after submit |

## Accessibility

- All form fields have associated labels
- Required fields marked with asterisk and `aria-required`
- Error messages linked via `aria-describedby`
- Focus order follows visual order
- Submit button disabled during submission
- Modal traps focus
