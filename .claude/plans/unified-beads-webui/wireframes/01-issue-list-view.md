## Component Breakdown

### Header/Navigation
- Logo + App name
- Primary navigation tabs
- Global search
- User menu

### Filter Bar
- Quick filter dropdowns:
  - Type (Bug, Feature, Task, Epic, Chore)
  - Status (Open, In Progress, Review, Done, Closed)
  - Priority (P1, P2, P3, P4)
  - Assignee (user dropdown with search)
  - Labels (multi-select)
- Add custom filter button
- Active filters display with clear option

## Responsive Breakpoints

### Desktop (>1024px)
- Full layout as shown above
- Side-by-side filters and search

### Tablet (768-1024px)
- Filters collapse to single row with overflow
- Cards maintain comfortable layout

### Mobile (<768px)
- Hamburger menu for navigation
- Stacked filter chips
- Cards simplified (hide labels, truncate title)
- Full-width search
