## Component Breakdown

### Header
- Title "Aging Items"
- Configure Thresholds button
- Current threshold display

### Severity Sections
- Critical section (ğŸ”´): Items exceeding critical threshold
- Warning section (ğŸŸ¡): Items between warning and critical
- Collapsible with count

### Aging Item Card
| Field | Description |
|-------|-------------|
| Age | Days since creation/started |
| RAG + ID + Title | Issue identification |
| Labels | Tag chips |
| Assignee | Owner with avatar |
| Last updated | Activity indicator |
| Epic | Parent epic link |
| Blocked by/Blocks | Dependency info |
| Threshold alert | Time until next severity |

### Filters
- Severity (Warning, Critical, Both)
- Assignee
- Epic
- Priority
- Updated date range

### Summary Bar
- Total aging items
- Critical/Warning breakdown
- Oldest item age
- Average age
