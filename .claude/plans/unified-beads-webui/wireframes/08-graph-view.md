## Component Breakdown

### Graph Canvas
- Pan and zoom enabled
- Click to select node
- Double-click to open issue
- Drag to pan

### Nodes
- Epics: Larger, distinct shape (rounded rectangle)
- Issues: Standard shape (rectangle)
- Color indicates priority/status
- Size can indicate children count

### Edges
- Solid: Dependency (blocks/blocked-by)
- Dashed: Parent/child relationship
- Arrow direction indicates flow

### Controls
- Zoom in/out/fit/center
- Layout selector
- Export and share options

### Tooltip/Popover
- Shows on hover or selection
- Quick info about node
- Actions: View Details, Dependencies

---

## Interactions

| Element | Action | Result |
|---------|--------|--------|
| Canvas | Drag | Pan view |
| Canvas | Scroll | Zoom in/out |
| Node | Click | Select and show popover |
| Node | Double-click | Open issue detail modal |
| Node | Drag | Move node (force layout) |
| Edge | Hover | Highlight relationship |
| [Fit] | Click | Fit all nodes in view |
| [Center] | Click | Center on selected or all |
| [Export PNG] | Click | Download graph as image |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `+` / `=` | Zoom in |
| `-` | Zoom out |
| `0` | Reset zoom |
| `f` | Fit to view |
| `c` | Center |
| `Arrow keys` | Pan |
| `Enter` | Open selected node |
| `Escape` | Deselect |

## Accessibility

- Keyboard navigation between nodes
- Screen reader announces node info
- Alternative table view available
- High contrast mode support
- Reduced motion respects prefers-reduced-motion

---

## Performance Notes

- Large graphs (>100 nodes) use virtualization
- Layout calculation is async with progress indicator
- Filtering reduces visible nodes for better performance
- WebGL renderer option for very large graphs
