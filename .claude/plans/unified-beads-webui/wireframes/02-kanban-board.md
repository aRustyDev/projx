## Component Breakdown

### Board Header
- Title "Board"
- Filter button (opens filter panel)
- Group by dropdown (Status, Priority, Assignee, Type)

### Column
- Status indicator + label + count
- Scrollable card container
- Optional "Add" button for Open column
- Drop zone highlighting on drag

### Card (KanbanCard)
- Priority indicator stripe (left edge color)
- Issue ID
- Title (truncated)
- Type badge
- Assignee avatar/name

### Card States
- Default: standard appearance
- Hover: slight elevation, show quick actions
- Dragging: elevated with shadow, cursor grabbing
- Drop target: column highlights

---

## Interactions

| Element | Action | Result |
|---------|--------|--------|
| Card | Click | Open issue detail modal |
| Card | Drag | Pick up card, show drop zones |
| Card | Drop on column | Update status via `bd update` |
| Column header | Click count | Scroll to top of column |
| [+ Add] | Click | Open create issue modal (status=Open) |
| Filter button | Click | Open filter panel |
| Group dropdown | Change | Regroup cards by selected field |

---

## Drag and Drop Behavior

### Visual Feedback
1. **Pickup**: Card elevates, becomes semi-transparent at origin
2. **Dragging**: Card follows cursor, columns highlight as valid targets
3. **Over column**: Drop zone indicator appears
4. **Drop**: Card animates to new position
5. **Cancel (Escape)**: Card returns to origin

### Status Update Flow
```
1. User drops card on new column
2. Optimistic update: Card appears in new column immediately
3. Background: `bd update [id] --status [newStatus]`
4. Success: Toast "Status updated"
5. Failure: Card returns to original column, error toast
```

## Grouping Options

| Group By | Columns |
|----------|---------|
| Status (default) | Open, In Progress, Review, Done, Closed |
| Priority | P1, P2, P3, P4, None |
| Assignee | One column per assignee + Unassigned |
| Type | Bug, Feature, Task, Epic, Chore |

---

## Responsive Behavior

### Desktop (>1024px)
- All columns visible, horizontal scroll if needed
- Full card details

### Tablet (768-1024px)
- Columns narrower
- Card titles truncated more aggressively
- Horizontal scroll enabled

### Mobile (<768px)
- Single column view with status tabs
- Swipe to change status tab
- Full-width cards

## Accessibility

- Cards are draggable via keyboard (Space to pick up, arrows to move, Enter to drop)
- Columns announced as lists with item counts
- Status changes announced to screen readers
- Focus visible on all interactive elements
