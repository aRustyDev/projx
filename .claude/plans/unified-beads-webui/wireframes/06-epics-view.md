## Component Breakdown

### Epic Card
- Expand/Collapse toggle (â–¶/â–¼)
- Epic icon (ðŸ“¦)
- Issue ID + Title
- Priority badge + Status badge
- Progress bar with count
- Owner + Target date + Blocker count

### Progress Bar
- Visual fill representing completion %
- Shows "X/Y tasks (Z%)"
- Color based on health:
  - Green: On track (>50% done with >50% time)
  - Amber: At risk
  - Red: Behind

### Breakdown Section
- Status distribution mini-chart
- Blocker alert if any blocked tasks
- Timeline info (start, target, remaining)

### Child List
- Nested epics with progress
- Direct task children
- Expand to see all children

### Actions
- View Details â†’ Opens issue detail modal
- Add Task â†’ Create modal with parent set
- Add Sub-epic â†’ Create epic modal with parent set

## Progress Calculations

```
Epic Progress = (Completed Children / Total Children) * 100

Where:
- Completed = status in [Done, Closed]
- Children = direct children + all nested epic children (recursive)

Blocker Count = count of children with status = Blocked or has blockedBy
```
