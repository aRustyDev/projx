## Component Breakdown

### Section Navigation
- Vertical list of sections
- Click to switch
- Current section highlighted

### Display Settings
- Theme (Light/System/Dark)
- Density (Compact/Standard/Wide)
- Time format
- Date format

### Thresholds
- Aging thresholds (Warning/Critical)
- RAG indicator colors (display only for MVP)
- Staleness threshold

### Columns
- Issue list columns (reorderable, toggleable)
- Epic list columns
- Kanban card fields (future)

### Keyboard
- Shortcut reference (read-only for MVP)
- Future: customizable bindings

### Data
- Connection info
- Real-time settings
- Cache management
