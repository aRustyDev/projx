## Component Breakdown

### Modal Header
- Title with issue ID and title
- Close button

### Current Issue Card
- Highlighted current issue
- RAG indicator, ID, title, priority, status

### Blocked By Section
- List of issues blocking this one
- Each shows its own blocking chain
- Click to navigate

### Blocking Section
- List of issues this one blocks
- Shows cascade effect

- Add Dependency â†’ Opens picker dialog
- View Graph â†’ Toggle to graph visualization
- Back to Issue â†’ Return to Issue Detail Modal
- Close â†’ Close modal

### Graph View
- Visual dependency graph
- Current issue highlighted

- Zoom and pan controls
