### Lead Time Scatter Plot
- X-axis: Completion date
- Y-axis: Lead time (days)
- Dots: Individual completed items
- Percentile lines: P50, P85, P95
- Hover for issue details

### Aging WIP Chart
- Horizontal bar chart
- Sorted by age (oldest first)
- Warning/Critical threshold lines
- Click to view issue

### Cumulative Flow Diagram
- Stacked area chart
- Status bands over time
- Shows flow and bottlenecks
- Hover for point-in-time counts

### Work Age Distribution
- Histogram of current WIP ages
- Buckets: 0-2d, 2-5d, 5-10d, 10-20d, >20d
- Warning zone highlighted

### Daily Throughput
- Bar chart of completions per day
- Rolling average line
- Click day for details

## Interactions

| Element | Action | Result |
|---------|--------|--------|
| Summary card | Click | Expand detail view |
| Chart point | Hover | Show tooltip with details |
| Chart item | Click | Open issue detail modal |
| [Configure] | Click | Open threshold config modal |
| Time scale | Change | Recalculate all metrics |
| Date range | Change | Filter data range |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|

| `r` | Refresh data |
| `t` | Toggle time scale |
| `d` | Change date range |
