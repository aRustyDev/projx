## Component Breakdown

### Auto-Calculate Toggle
- Checkbox to enable/disable
- Shows percentile info when enabled
- Recalculate button
- Disables manual inputs when on

### Threshold Inputs
- Integer input for value
- Scale dropdown (Hours/Days/Weeks)
- Validation feedback

### Preview Section
- Real-time calculation
- Shows counts per severity
- Lists affected issue IDs
- Updates on input change

### Actions
- Reset to Defaults
- Cancel
- Save

---

## Validation Rules

| Rule | Error Message |
|------|---------------|
| Warning >= Critical | "Warning must be less than critical threshold" |
| Value <= 0 | "Threshold must be greater than 0" |
| Value > 365 days | "Threshold cannot exceed 1 year" |

---

## Interactions

| Element | Action | Result |
|---------|--------|--------|
| Auto-calculate toggle | Check | Disable manual inputs, calculate |
| [Recalculate] | Click | Refresh auto-calculated values |
| Value input | Change | Update preview |
| Scale dropdown | Change | Convert and update preview |
| Preview issue ID | Click | Open issue detail |
| [Reset to Defaults] | Click | Reset to 7d/14d |
| [Save] | Click | Save and close |

---

## Default Values

| Threshold | Default | Auto-Calculate Percentile |
|-----------|---------|---------------------------|
| Warning | 7 days | P75 of lead time |
| Critical | 14 days | P90 of lead time |
