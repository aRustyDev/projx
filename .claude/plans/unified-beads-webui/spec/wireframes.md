### View-Specific Adaptations

**Kanban Board:**
- Mobile: Single column, swipe to change status
- Tablet: 2 columns visible, horizontal scroll
- Desktop: All columns visible

**Metrics Dashboard:**
- Mobile: Stack all cards vertically
- Tablet: 2-column grid
- Desktop: Full dashboard layout

**Gantt Chart:**
- Mobile: List view only (no timeline)
- Tablet: Compressed timeline (week view only)
- Desktop: Full timeline with zoom controls

#### Keyboard Navigation

All interactive elements are focusable and operable via keyboard:

1. **Tab order**: Logical left-to-right, top-to-bottom
2. **Skip links**: "Skip to main content" link at page start
3. **Focus trapping**: Modals trap focus within
4. **Escape**: Closes modals and dropdowns
5. **Arrow keys**: Navigate within lists and menus
